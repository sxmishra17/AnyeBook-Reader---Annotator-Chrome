Privacy Policy

Any eBook Reader & Annotator — Chrome Extension

ZERO DATA COLLECTION

In short: This extension does not collect, store, transmit, or share any personal data whatsoever. Everything happens locally on your device.

1. Information We Collect

None. This extension does not collect any information from you. It does not track your browsing activity, file usage, reading habits, or any other personal data.

2. Data Storage

All data created while using the extension (notes, highlights, annotations) is stored exclusively in your browser's local storage on your device. This data:

  • Never leaves your device
  • Is never transmitted to any server
  • Is never shared with any third party
  • Is fully under your control and can be deleted at any time

3. Network Requests

This extension makes zero network requests. It does not connect to any external servers, APIs, or services. All file processing (PDF, EPUB, MOBI, AZW3, DOCX) is performed entirely on your local device using bundled open-source libraries.

4. Permissions

The extension requests the following browser permissions, used solely for its core functionality:

  • activeTab / tabs: Used only to capture screenshots for the snapshot/annotation feature
  • storage: Used to save your notes and preferences locally in your browser
  • downloads: Used to export your notes as DOCX, PDF, or TXT files to your device
  • host_permissions (<all_urls>): Required by Chrome for the tab screenshot API used in the snapshot feature. No web page content is accessed or collected.

5. Third-Party Services

This extension does not use any third-party analytics, tracking, advertising, or data collection services.

6. Children's Privacy

This extension does not knowingly collect any data from children or any other users. No data is collected from anyone.

7. Changes to This Policy

If this privacy policy is updated, the changes will be reflected on this page with an updated effective date.

8. Contact

If you have questions about this privacy policy, you can reach us through the extension's Chrome Web Store listing or by opening an issue on our GitHub repository.
